**版本:** V2.2

## 1. 背景与目标

### 1.1 背景
当前 V2.1 版本中，三种计算模式存在以下问题：
1. 选择水/水蒸气介质时，流量单位仍显示 Nm³/h，不符合工程习惯
2. 模式 1 选择混合介质后，无法输入组分百分比
3. 模式 2/3 缺少单一/混合介质选项，无法进行混合介质计算

### 1.2 目标
- ✅ 实现水/水蒸气介质时流量单位自动切换到 T/h
- ✅ 支持混合介质组分输入（N₂、O₂、H₂、CO₂、H₂O）
- ✅ 提供组分归一化功能
- ✅ 统一三种模式的介质选择体验

---

## 2. 需求详情

### 2.1 需求 1: 水/水蒸气流量单位自动切换

**需求描述:**
当用户选择介质为"水/水蒸气 (H₂O)"时，流量单位自动切换为"T/h"并禁用 Nm³/h 选项。

**交互逻辑:**
```
用户选择介质 → 判断是否为 H₂O → 是则切换流量单位为 T/h
用户切换介质 → 判断是否离开 H₂O → 是则恢复 Nm³/h 选项
```

**影响范围:**
- `Mode1Form.vue` - 冷边、热边
- `Mode2Form.vue` - 涡轮入口、换热器热边
- `Mode3Form.vue` - 涡轮入口

---

### 2.2 需求 2: 混合介质组分输入（模式 1）

**需求描述:**
模式 1 选择"混合介质"后，展开组分输入界面，支持输入 5 种介质的百分比。

**UI 结构:**
```
介质类型选择：[单一介质 ▼] [混合介质 ▼]
              ↓ (选择混合介质)
┌─────────────────────────────────────────────────────┐
│ 混合介质组分配置                                      │
├─────────────────────────────────────────────────────┤
│ 百分比类型：○ 摩尔百分比  ○ 质量百分比               │
├─────────────────────────────────────────────────────┤
│ N₂:  [____] %   O₂:  [____] %   H₂:  [____] %       │
│ CO₂: [____] %   H₂O: [____] %                       │
├─────────────────────────────────────────────────────┤
│ 当前合计：XX.XX%    [🔄 归一化]                      │
└─────────────────────────────────────────────────────┘
```

**功能要求:**
| 功能 | 描述 |
|------|------|
| 组分输入 | 5 种介质，支持小数，精确到小数点后 2 位 |
| 百分比类型 | 摩尔百分比 / 质量百分比，单选切换 |
| 实时合计 | 实时显示当前输入总和 |
| 归一化按钮 | 点击后将各组分按比例缩放至合计 100% |
| 校验提示 | 提交时若合计≠100%，提示用户先归一化 |

**数据格式:**
```javascript
{
  medium_type: 'mix',
  mix_composition: { N2: 0.21, O2: 0.79, H2: 0, CO2: 0, H2O: 0 },
  composition_type: 'mole'  // 或 'mass'
}
```

## 3. 技术方案

### 3.1 组件复用

创建可复用的混合介质组分输入组件：
```
frontend/src/components/common/MixCompositionInput.vue
```

**Props:**
| 属性 | 类型 | 默认值 | 描述 |
|------|------|--------|------|
| modelValue | Object | - | 组分数据 |
| prefix | String | '' | 字段前缀（用于表单绑定） |

**Emits:**
| 事件 | 参数 | 描述 |
|------|------|------|
| update:modelValue | Object | 组分数据变化 |

### 3.2 状态管理

每个表单组件内部维护：
```javascript
const form = reactive({
  cold: {
    medium_type: 'single',  // 'single' | 'mix'
    medium: 'N2',
    mix_composition: { N2: 20, O2: 80, H2: 0, CO2: 0, H2O: 0 },
    composition_type: 'mole',  // 'mole' | 'mass'
    flow_rate: 1000,
    flow_unit: 'Nm3/h',
    // ...
  },
  // ...
})
```

### 3.3 归一化逻辑

```javascript
function normalize() {
  const total = Object.values(composition).reduce((a, b) => a + b, 0)
  if (total === 0 || total === 100) return
  const factor = 100 / total
  for (const key in composition) {
    composition[key] = Math.round(composition[key] * factor * 100) / 100
  }
}
```

### 3.4 水/蒸汽单位切换逻辑

```javascript
watch(() => form.cold.medium, (newMedium) => {
  if (newMedium === 'H2O') {
    form.cold.flow_unit = 'T/h'
  }
})
```
