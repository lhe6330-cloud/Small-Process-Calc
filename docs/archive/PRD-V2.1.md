## 1. 项目概述

### 1.1 产品定位

PDS CALC V2.1 是在 V2.0 基础上的优化版本，专注于：
- **简化用户界面** - 移除冗余的模式 4/5 标签页
- **优化交互流程** - 分离器/涡轮设计集成到计算结果面板
- **修复已知问题** - PDF/Excel 导出功能修复
- **增强功能** - 支持多分离器设计（最多 5 个）

### 1.2 版本变更

| 变更项 | V2.0 | V2.1 | 说明 |
|--------|------|------|------|
| 模式 4 标签页 | ✅ 有 | ❌ 移除 | 功能集成到计算结果面板 |
| 模式 5 标签页 | ✅ 有 | ❌ 移除 | 功能集成到计算结果面板 |
| 添加分离器按钮 | ✅ 有 | ✅ 优化 | 支持多分离器（最多 5 个） |
| 涡轮一维设计按钮 | ✅ 有 | ❌ 移除 | 保留弹窗功能 |
| 分离器数据来源 | 全部节点 | 按模式隔离 | 模式 1/2/3 独立 |
| PDF/Excel 导出 | ❌ 报错 | ✅ 修复 | 支持 V2.0 功能 |

## 2. 功能需求

### 2.1 功能架构


### 3.2 分离器设计优化

#### 3.2.1 数据来源按模式隔离（方案 A）

**模式 1 数据来源：**
| 选项 | 显示名称 | 节点 ID |
|------|----------|---------|
| 1 | 模式 1 - 冷边入口 | mode1_cold_inlet |
| 2 | 模式 1 - 换热器出口 | mode1_hx_out |
| 3 | 模式 1 - 涡轮出口 | mode1_turbine_out |

**模式 2 数据来源：**
| 选项 | 显示名称 | 节点 ID |
|------|----------|---------|
| 1 | 模式 2 - 涡轮入口 | mode2_turbine_in |
| 2 | 模式 2 - 涡轮出口 | mode2_turbine_out |
| 3 | 模式 2 - 换热器出口 | mode2_hx_out |

**模式 3 数据来源：**
| 选项 | 显示名称 | 节点 ID |
|------|----------|---------|
| 1 | 模式 3 - 涡轮入口 | mode3_turbine_in |
| 2 | 模式 3 - 涡轮出口 | mode3_turbine_out |

**实现逻辑：**
```javascript
// 根据当前模式过滤数据来源
const getSeparatorNodes = (currentMode) => {
  if (currentMode === 'mode1') {
    return ['mode1_cold_inlet', 'mode1_hx_out', 'mode1_turbine_out']
  }
  if (currentMode === 'mode2') {
    return ['mode2_turbine_in', 'mode2_turbine_out', 'mode2_hx_out']
  }
  if (currentMode === 'mode3') {
    return ['mode3_turbine_in', 'mode3_turbine_out']
  }
}
```

#### 3.2.2 多分离器支持

**功能描述：**
- 每次点击"📐 添加分离器"按钮，新增一个分离器计算区域（可多个并列）
- 最多支持 5 个分离器
- 每个分离器的数据来源独立选择（按当前模式过滤）
- 每个分离器的设计参数独立配置
- 每个分离器的计算结果独立显示

**界面设计：**

#### 3.2.3 分离器计算流程

```
1. 用户运行模式 1/2/3 计算 → 得到完整流程参数
2. 计算结果面板自动展开"流程节点分离器设计"区域
3. 点击 [📐 添加分离器] 按钮 → 新增一个分离器计算区域
4. 选择数据来源（下拉菜单，按当前模式过滤）
5. 点击 [📥 获取数据] → 自动带入该节点的工况参数
6. 用户补充设计参数（液滴粒径、长径比等）
7. 点击 [🚀 开始计算] → 计算分离器尺寸
8. 计算结果显示在分离器区域内
9. 可继续点击 [➕ 添加分离器] 添加更多（最多 5 个）
```

```javascript
const getTurbineSource = (currentMode) => {

```

```
5. 点击 [🚀 开始计算] → 计算涡轮一维设计
6. 计算结果显示在涡轮设计区域内
```

### 3.4 报告导出优化

#### 3.4.1 导出内容规则（方案 C：自动检查）

| 计算情况 | PDF/Excel 内容 |
|----------|----------------|
| 只计算了模式 1/2/3 | 只有模式 1/2/3 基础数据 |
| 计算了模式 1/2/3 + 分离器（1 个或多个） | 模式 1/2/3 数据 + 所有分离器数据 |
| 计算了模式 1/2/3 + 涡轮设计 | 模式 1/2/3 数据 + 涡轮设计数据 |
| 计算了模式 1/2/3 + 分离器 + 涡轮设计 | 模式 1/2/3 数据 + 所有分离器数据 + 涡轮设计数据 |

**实现逻辑：**
```javascript
const exportPDF = async () => {
  const exportData = {
    ...inputData,  // 模式 1/2/3 基础数据
    separators: separators.value,  // 自动包含所有分离器（如有）
    turbine_1d: turbine1d.value,   // 自动包含涡轮设计（如有）
  }
  
  // 调用后端 API 导出
  await fetch('/api/export/pdf/' + currentMode, {
    method: 'POST',
    body: JSON.stringify(exportData)
  })
}
```

#### 3.4.2 导出接口

| 接口 | 方法 | 说明 |
|------|------|------|
| `/api/export/pdf/mode1` | POST | 导出模式 1 PDF（含 V2.0 功能） |
| `/api/export/pdf/mode2` | POST | 导出模式 2 PDF（含 V2.0 功能） |
| `/api/export/pdf/mode3` | POST | 导出模式 3 PDF（含 V2.0 功能） |
| `/api/export/excel/mode1` | POST | 导出模式 1 Excel（含 V2.0 功能） |
| `/api/export/excel/mode2` | POST | 导出模式 2 Excel（含 V2.0 功能） |
| `/api/export/excel/mode3` | POST | 导出模式 3 Excel（含 V2.0 功能） |

#### 3.4.3 Request Body

```json
{
  "cold_side": {...},

  "turbine": {...},
  "separators": [
    {
      "node_id": "mode1_hx_out",

    {
      "node_id": "mode1_turbine_out",

      "result": {...}
```
