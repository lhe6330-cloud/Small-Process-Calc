## 测试步骤

### 步骤 1: 刷新浏览器
由于前端使用了 Vite，文件修改后会自动热重载。如果没有自动刷新，请手动刷新浏览器。

### 步骤 2: 执行计算
1. 打开 http://localhost:3000
2. 选择"模式 1: 先加热再膨胀"
3. 填写或确认默认参数
4. 点击"🚀 开始计算"

### 步骤 3: 测试导出
计算完成后，在结果面板底部会看到两个按钮：

```
┌─────────────────────────────────────┐
│  📊 计算结果                        │
├─────────────────────────────────────┤
│  [统计数据显示...]                   │
├─────────────────────────────────────┤
│  [📄 导出 PDF]  [📊 导出 Excel]      │
└─────────────────────────────────────┘
```

1. 点击"📄 导出 PDF"
   - 应该下载 `report_PDF_XXXXXXXXXXX.pdf`
   - 文件大小约 50-60 KB
   - 用 Adobe Reader 或浏览器打开查看

2. 点击"📊 导出 Excel"
   - 应该下载 `report_Excel_XXXXXXXXXXX.xlsx`
   - 文件大小约 5-6 KB
   - 用 Excel 打开查看

## 常见问题

### Q1: 点击导出按钮无反应
**原因**: localStorage 中没有输入数据  
**解决**: 先执行一次计算，然后再导出

### Q2: 导出失败提示"未找到输入数据"
**原因**: 页面刷新后 localStorage 数据丢失  
**解决**: 重新填写参数并计算

### Q3: 中文显示乱码（PDF）
**原因**: 系统未找到中文字体  
**解决**: 确保 Windows 系统有微软雅黑或宋体

### Q4: 下载的文件打不开
**原因**: 可能是网络问题导致文件不完整  
**解决**: 检查文件大小，重新导出
